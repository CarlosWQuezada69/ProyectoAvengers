# Proyecto Avengers

Sistema integral de gestión comercial con landing page para clientes y panel administrativo.

## Descripción

**Proyecto Avengers** es una aplicación web que combina un landing page corporativo con un panel de administración. El landing page muestra los productos del negocio con enlaces a redes sociales, mientras que el panel administrativo permite gestionar el catálogo de productos y monitorear estadísticas de visualización.

## Características

### Landing Page (Clientes)
- Catálogo de productos en tiempo real
- Integración con redes sociales del negocio
- Diseño responsivo y moderno

### Panel de Administración
- CRUD completo de productos
- Estadísticas de visualización por producto
- Dashboard con métricas en tiempo real

## Tecnologías propuestas

- **Frontend:** React / Next.js / Vue.js
- **Backend:** Node.js / Python
- **Base de datos:** PostgreSQL / MongoDB
- **Autenticación:** JWT / NextAuth
